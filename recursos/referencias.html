Referencias y recursos adicionales

Libros de texto

Textos principales (en inglés)

1. Introduction to Modern Statistics (IMS)

  • Autores: Mine Çetinkaya-Rundel y Johanna Hardin
  • Editorial: OpenIntro (2023)
  • Acceso: Gratis online
  • Descripción: Texto moderno de estadística con enfoque en simulación y bootstrap. Excelente para aprender estadística inferencial.
  • Nivel: Introductorio

2. OpenIntro Statistics

  • Autores: David Diez, Mine Çetinkaya-Rundel, Christopher Barr
  • Editorial: OpenIntro (4ta edición)
  • Acceso: Gratis online
  • Descripción: Texto completo de estadística introductoria con muchos ejemplos y ejercicios.
  • Nivel: Introductorio

3. Statistical Inference via Data Science: A ModernDive into R

  • Autores: Chester Ismay y Albert Y. Kim
  • Acceso: Gratis online
  • Descripción: Enfoque moderno usando tidyverse. Excelente para aprender R y estadística simultáneamente.
  • Nivel: Introductorio

4. R for Data Science (r4ds)

  • Autores: Hadley Wickham, Mine Çetinkaya-Rundel, Garrett Grolemund
  • Editorial: O’Reilly (2da edición, 2023)
  • Acceso: Gratis online
  • Descripción: Referencia esencial para análisis de datos con R y tidyverse.
  • Nivel: Introductorio a intermedio

5. Introduction to Statistical Learning (ISL)

  • Autores: Gareth James, Daniela Witten, Trevor Hastie, Robert Tibshirani
  • Editorial: Springer (2da edición, 2021)
  • Acceso: Gratis online
  • Descripción: Introducción accesible al aprendizaje estadístico. Incluye laboratorios en R.
  • Nivel: Intermedio

Textos de referencia avanzada

6. Applied Linear Statistical Models

  • Autores: Kutner, Nachtsheim, Neter, Li
  • Editorial: McGraw-Hill (5ta edición)
  • Descripción: Texto comprehensivo sobre regresión y ANOVA.
  • Nivel: Intermedio a avanzado

7. The Elements of Statistical Learning

  • Autores: Trevor Hastie, Robert Tibshirani, Jerome Friedman
  • Editorial: Springer (2da edición)
  • Acceso: Gratis online
  • Descripción: Texto avanzado sobre aprendizaje estadístico.
  • Nivel: Avanzado

Libros en español

8. Estadística para Investigadores

  • Autores: George W. Snedecor, William G. Cochran
  • Descripción: Clásico de estadística aplicada, disponible en español.
  • Nivel: Intermedio

9. Introducción a la Estadística

  • Autores: Mario Triola

  • Descripción: Texto introductorio en español, ampliamente usado.
  • Nivel: Introductorio

Tablas estadísticas

Tablas de distribuciones

En R, no necesitas tablas físicas (todo se calcula con funciones), pero si las prefieres:

  • Tabla Z (normal estándar): Buscar “standard normal table PDF”
  • Tabla t: Buscar “t distribution table PDF”
  • Tabla F: Buscar “F distribution table PDF”
  • Tabla chi-cuadrado: Buscar “chi-square table PDF”

Usar R en lugar de tablas

# Valor crítico z para 95% de confianza (bilateral)
qnorm(0.975)  # 1.96

# Valor crítico t para 95% con 20 grados de libertad
qt(0.975, df = 20)  # 2.086

# Probabilidad: P(Z < 1.96)
pnorm(1.96)  # 0.975
