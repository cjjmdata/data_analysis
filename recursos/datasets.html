Catálogo de datasets

Introducción

Este catálogo presenta los datasets disponibles para prácticas, tareas y proyectos del curso. Todos los datasets están preparados para análisis estadístico y documentados con descripciones de variables.

Datasets incluidos en R

mtcars - Motor Trend Car Road Tests

Descripción: Datos de rendimiento y características de 32 automóviles (1973-74).

Variables: 11 variables, 32 observaciones

Variables principales: - mpg: Millas por galón - cyl: Número de cilindros - hp: Caballos de fuerza - wt: Peso (1000 lbs) - qsec: Tiempo en 1/4 de milla - am: Transmisión (0=automática, 1=manual)

Uso típico: Regresión lineal, comparación de grupos

Cómo cargar:

data(mtcars)
head(mtcars)

iris - Edgar Anderson’s Iris Data

Descripción: Mediciones de flores iris de 3 especies diferentes.

Variables: 5 variables, 150 observaciones (50 por especie)

Variables: - Sepal.Length: Longitud del sépalo (cm) - Sepal.Width: Ancho del sépalo (cm) - Petal.Length: Longitud del pétalo (cm) - Petal.Width: Ancho del pétalo (cm) - Species: Especie (setosa, versicolor, virginica)

Uso típico: Análisis exploratorio, ANOVA, clasificación

Cómo cargar:

data(iris)
head(iris)

diamonds - Prices of over 50,000 round cut diamonds

Descripción: Precio y atributos de ~54,000 diamantes (paquete ggplot2).

Variables: 10 variables, 53,940 observaciones

Variables principales: - price: Precio en dólares USD - carat: Peso del diamante - cut: Calidad del corte (Fair, Good, Very Good, Premium, Ideal) - color: Color del diamante (D mejor, J peor) - clarity: Claridad (I1 peor, IF mejor) - x, y, z: Dimensiones en mm

Uso típico: Regresión múltiple, variables categóricas

Cómo cargar:

library(ggplot2)
data(diamonds)
head(diamonds)

mpg - Fuel economy data

Descripción: Economía de combustible de 234 modelos de vehículos (1999-2008).

Variables: 11 variables, 234 observaciones

Variables principales: - manufacturer: Fabricante - model: Modelo - displ: Desplazamiento del motor (litros) - year: Año de fabricación - cyl: Número de cilindros - cty: Millas por galón en ciudad - hwy: Millas por galón en carretera - class: Tipo de vehículo

Uso típico: Regresión, análisis comparativo

Cómo cargar:

library(ggplot2)
data(mpg)
head(mpg)

Datasets del curso

Los siguientes datasets estarán disponibles en la carpeta data/ del repositorio

estudiantes.csv

Descripción: Datos ficticios de rendimiento académico de estudiantes.

Variables: 8 variables, 200 observaciones

Variables: - id: Identificador del estudiante - edad: Edad en años - genero: Género (M/F) - horas_estudio: Horas de estudio semanal - faltas: Número de faltas - calificacion_parcial1: Nota del primer parcial (0-100) - calificacion_parcial2: Nota del segundo parcial (0-100) - calificacion_final: Nota final (0-100)

Uso típico: Regresión múltiple, pruebas t, ANOVA

Cómo cargar:

library(readr)
estudiantes <- read_csv("data/estudiantes.csv")

salarios.csv

Descripción: Datos de salarios y características de empleados.

Variables: 7 variables, 500 observaciones

Variables: - id: Identificador - salario: Salario anual (miles de USD) - experiencia: Años de experiencia - educacion: Años de educación - genero: Género (M/F) - departamento: Departamento (Ventas, IT, Marketing, HR) - ciudad: Ciudad (A, B, C)

Uso típico: Regresión múltiple con variables dummy, análisis salarial

Cómo cargar:

salarios <- read_csv("data/salarios.csv")

experimento_farmacos.csv

Descripción: Resultados de un experimento con 3 tratamientos farmacológicos.

Variables: 4 variables, 90 observaciones

Variables: - paciente_id: ID del paciente - tratamiento: Tratamiento (A, B, C) - edad: Edad del paciente - respuesta: Variable de respuesta (reducción de síntomas, 0-100)

Uso típico: ANOVA unidireccional, pruebas post-hoc

Cómo cargar:

farmacos <- read_csv("data/experimento_farmacos.csv")

ventas_tiendas.csv

Descripción: Ventas mensuales de tiendas con diferentes características.

Variables: 6 variables, 120 observaciones

Variables: - tienda_id: ID de la tienda - mes: Mes del año (1-12) - ventas: Ventas en miles de USD - publicidad: Gasto en publicidad (miles de USD) - zona: Zona geográfica (Norte, Sur, Este, Oeste) - tamaño: Tamaño de la tienda (Pequeña, Mediana, Grande)

Uso típico: Regresión, ANOVA bidireccional

Cómo cargar:

ventas <- read_csv("data/ventas_tiendas.csv")

satisfaccion_clientes.csv

Descripción: Encuesta de satisfacción de clientes.

Variables: 5 variables, 300 observaciones

Variables: - cliente_id: ID del cliente - satisfaccion: Nivel de satisfacción (1-5) - tiempo_espera: Tiempo de espera en minutos - calidad_servicio: Calificación de calidad (1-10) - tipo_servicio: Tipo (Estándar, Premium)

Uso típico: Regresión, datos categóricos ordinales

encuesta_preferencias.csv

Descripción: Datos de encuesta sobre preferencias de productos.

Variables: 4 variables, 400 observaciones

Variables: - id: Identificador - edad_grupo: Grupo de edad (18-25, 26-35, 36-50, 50+) - genero: Género (M/F) - preferencia: Producto preferido (A, B, C, D)

Uso típico: Tablas de contingencia, prueba chi-cuadrado

Cómo cargar:

encuesta <- read_csv("data/encuesta_preferencias.csv")

Guía de uso de datasets

Exploración inicial

Cuando trabajas con un dataset nuevo, sigue estos pasos:

# 1. Cargar datos
datos <- read_csv("ruta/al/archivo.csv")

# 2. Ver primeras filas
head(datos)

# 3. Estructura del dataset
str(datos)

# 4. Resumen estadístico
summary(datos)

# 5. Dimensiones
dim(datos)

# 6. Nombres de variables
names(datos)

# 7. Verificar datos faltantes
sum(is.na(datos))
colSums(is.na(datos))

# 8. Tipos de variables
sapply(datos, class)

Limpieza de datos

library(tidyverse)

# Eliminar filas con NA
datos_limpio <- datos %>% drop_na()

# Reemplazar NA con la media (solo variables numéricas)
datos$variable[is.na(datos$variable)] <- mean(datos$variable, na.rm = TRUE)

# Convertir variables a factor
datos$categoria <- as.factor(datos$categoria)

# Renombrar variables
datos <- datos %>%
  rename(nueva_nombre = viejo_nombre)

Consideraciones éticas

Al usar datasets reales

  1. Privacidad: No uses datasets con información personal identificable sin anonimizar
  2. Consentimiento: Verifica que los datos fueron recolectados éticamente
  3. Sesgos: Reconoce posibles sesgos en los datos
  4. Interpretación: Ten cuidado con interpretaciones que puedan perpetuar discriminación

Citar datasets

Siempre cita la fuente de los datos:

Ejemplo:
Datos obtenidos de UCI Machine Learning Repository:
Dua, D. and Graff, C. (2019). UCI Machine Learning Repository
[http://archive.ics.uci.edu/ml]. Irvine, CA: University of California,
School of Information and Computer Science.

Crear tus propios datasets

Simulación de datos en R

# Generar datos normales
set.seed(123)
datos_sim <- data.frame(
  x = rnorm(100, mean = 50, sd = 10),
  y = rnorm(100, mean = 100, sd = 15)
)

# Generar datos con relación lineal
set.seed(123)
n <- 100
x <- runif(n, 0, 10)
y <- 2 + 3*x + rnorm(n, 0, 2)
datos_regresion <- data.frame(x, y)

# Generar datos categóricos
set.seed(123)
datos_cat <- data.frame(
  grupo = sample(c("A", "B", "C"), 100, replace = TRUE),
  valor = rnorm(100, 50, 10)
)

TipConsejo para proyectos

Para el proyecto final, elige un dataset que:

  1. Te interese personalmente
  2. Tenga suficientes observaciones (n > 50)
  3. Tenga variables relevantes para análisis
  4. Esté bien documentado
  5. Permita aplicar múltiples técnicas del curso
